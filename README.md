HyperCommander
==============

Total Commander 를 모방한 파일 관리자. 

목표
====

1. Qt + QtitanDataGrid 를 이용하여 토탈 커맨더의 기능 구현( 학습용 )
2. Total Commander 와 같은 파일관리자와 같은 UI 를 위해 QtitanDataGrid Row/Cell UI 커스터마이즈

기능
====

1. 파일 동작
1.1 이름 변경 : 인라인 이름변경( F2, 마우스 클릭 )
